Grape Embryo
=================

A [Grape](http://github.com/ruby-grape/grape) API mounted on Rack.

See
---

There's a deployed [grape-embryo on my Website](http://grape-embryo.wilsonsilva.net).

Run
---

```
$ bundle install
$ rackup
```
